# 调查员开局指南

> 创建日期: 2026-05-08
> 最后更新: 2026-05-08
> 来源数: 3
> 状态: ✅ 已填充

## 概述

这是一张面向玩家的开局导航页，用来把“创建调查员 -> 分配技能 -> 看角色卡 -> 开始进行检定 -> 获得成长标记”这条路径串起来。它的目标不是替代各个细分规则页，而是帮助玩家在第一次建卡和第一次上桌时，快速知道自己该按什么顺序理解规则。

---

## 详细内容

### 1. 先知道你在扮演什么

《克苏鲁的呼唤》里的玩家角色叫**调查员**。他们通常不是战斗机器，而是靠调查、判断、社交、知识、运气与偶尔的拼命来活下去的人。游戏没有传统意义上的“打怪升级主线”，更多是在恐怖情境中搜集线索、做判断、承担代价。

如果你想先获得对玩法的直觉，可以先读 [[coc-overview.md]]。

### 2. 创建角色时先做什么

建卡的大顺序是：

1. 生成属性与幸运值  
2. 根据年龄调整属性  
3. 选择职业  
4. 计算并分配本职技能点  
5. 计算并分配兴趣技能点  
6. 记录背景条目与关键连接  
7. 计算衍生属性、半值与五分之一值

完整流程见 [[investigator-creation.md]]，这里更关注“玩家第一次建卡最容易卡住的地方”。

### 3. 属性到底决定什么

八项属性不是摆设，它们会直接影响你的角色卡结构：

- **INT** 决定兴趣技能点，通常为 `INT × 2`
- **EDU** 常常参与职业技能点计算
- **POW** 决定起始理智值，并影响魔法值
- **STR / CON / SIZ / DEX** 会继续影响伤害加值、体格、生命值、移动速度和战斗表现

幸运值单独掷出，不属于八大属性，但在游戏中很常用。

更详细的属性解释见 [[attribute-system.md]]；HP、MOV、伤害加值、体格、幸运等衍生结果见 [[derived-attributes.md]]。

### 4. 职业怎么选才实用

职业最重要的功能，不是限制你“平时只能干什么”，而是决定：

- 你的**本职技能列表**
- 你的**本职技能点公式**
- 你的**信用评级范围**

换句话说，职业决定你的角色开局“擅长什么”，但不会把你锁死。你仍然可以用兴趣技能点把角色塑造成偏门、混搭或不那么标准的样子。

第一次建卡时，最省心的办法通常是：

- 先想人物概念
- 再找最贴近概念的职业
- 最后让职业技能为概念服务

职业详情见 [[occupation-system.md]]。

### 5. 技能点怎么分才不容易后悔

技能点分成两类：

- **本职技能点**：按职业公式计算，只能分给职业列出的技能与信用评级
- **兴趣技能点**：通常为 `INT × 2`，可分给你想强化的大多数技能

分配时要记住两件事：

- 技能点是**加在基础值上**的，不是从零开始写
- 没分完的本职技能点会消失

技能值的大致手感可以这样理解：

- `20%-49%`：会一点，有爱好者水平
- `50%-74%`：可以靠它吃饭
- `75%-89%`：专家
- `90%+`：顶尖人物

更完整的技能说明和技能专攻规则见 [[skill-system.md]]。

### 6. 半值和五分之一值为什么最好先算

在 CoC 7e 里，很多检定不是只看一个完整值，还会频繁用到：

- **半值**：困难成功
- **五分之一值**：极限成功

这些数值最好在建卡结束时就全部预先算好，填在角色卡上。否则一到紧张场景就要边算边玩，节奏会很容易散掉。

相关说明见 [[half-and-fifth-values.md]]。

### 7. 上桌后，玩家最常做什么检定

最常见的是**技能检定**。基本流程很简单：

1. 先说明你想达成什么
2. 守秘人决定用什么技能或属性，以及难度
3. 掷 `1D100`
4. 若结果小于等于目标值，则成功

常规情况下看完整值；更难时看半值；极端困难时看五分之一值。失败不一定意味着“什么都没发生”，也可能意味着事情朝糟糕方向推进。

技能检定、孤注一掷、幸运检定等内容见 [[skill-check.md]]。  
如果你想看幸运值怎么在实战里帮你补救，则见 [[luck-and-spending-luck.md]]。

### 8. 角色卡上哪些地方要特别留意

第一次上桌前，至少确认这些项目已经写清楚：

- 八项属性与幸运
- HP、MP、SAN、MOV、伤害加值、体格
- 各技能完整值、半值、五分之一值
- 职业与信用评级
- 武器和对应技能专攻
- 背景条目与关键连接

其中背景条目不只是风味文本，它会影响自救、理智恢复和角色长期变化。相关内容可以配合 [[sanity-and-recovery-guide.md]] 阅读。

### 9. 建卡完成前检查清单

如果你现在正准备把角色卡收尾，上桌前可以快速过这一遍：

- 属性和幸运都已经写好
- 职业已经选定，本职技能点已经分完
- 兴趣技能点已经分完
- HP、MP、SAN、MOV、伤害加值、体格都已经算好
- 常用技能的完整值、半值、五分之一值都已经写好
- 武器和对应技能专攻已经记上
- 信用评级已经落在职业允许范围内
- 背景条目与关键连接已经写上

如果这几项都齐了，你的角色卡通常就已经能顺利开跑了。

### 10. 游戏里怎么变强

CoC 的成长不是传统的线性升级，而是“你在游戏里成功用过什么，幕间才可能成长什么”。

只要某项技能在游戏中**成功使用**，通常就可以在角色卡上打一个成长标记。到了模组结束或幕间成长阶段，再为这些技能做成长判定。不是用得越多涨得越多，而是“有成功使用过，就有成长机会”。

成长机制详见 [[experience-reward.md]]。

## 核心规则/要点

- 玩家开局最重要的是先完成建卡顺序，而不是背完整技能表。
- 属性决定技能点、衍生属性和角色的行为轮廓。
- 职业决定本职技能、技能点公式和信用评级范围。
- 技能点是加在基础值上的，半值与五分之一值要提前算好。
- 上桌最常用的是技能检定、幸运检定和理智检定。
- 成长依赖成功使用技能后的成长标记，而不是传统升级。
