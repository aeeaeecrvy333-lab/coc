## 概述

本文汇总《怪物之锤》第一卷与第二卷附录A中的神话名称发音表，供守秘人在念诵怪物、种族与神祇名号时快速查阅。原附录主要给出 Chaosium 侧的英语读法，适合作为跑团时的统一口径，而非唯一标准。

---

## 详细内容

### 发音说明

《怪物之锤》第一卷附录在表前提供了一组简要音标说明。译者附注如下：

- `’ = /ʔ/`：按汉语拼音中的声门塞音处理。
- `A = /æ/`，有时也可能读作 `/ʌ/`。
- `AH = /ɑː/`
- `AE = /eɪ/`
- `E = /e/`
- `EH = /e/`
- `EE = /iː/`
- `I = /ɪ/`，词尾有时接近 `/i/`
- `IH = /ɪ/`，词尾有时接近 `/i/`
- `IGH = /aɪ/`
- `O = /ɔ/` 或 `/ɒ/`
- `AU = /ɔː/`
- `OE = /oʊ/`
- `U = /ʊ/` 或 `/ʌ/`
- `UH = /ʊ/` 或 `/ə/`
- `OO = /uː/`
- `Y = /j/`
- `OI = /ɔɪ/`

第二卷另有一条补充说明：`/x/` 在英语中罕见，近似汉语拼音中的 `h`。

- 本附录提供的是跑团朗读时的统一参考发音，不是设定层面的硬性唯一标准。
- 第一卷偏向怪物、种族与相关名词；第二卷偏向神祇与独特存在。
- 某些词条保留了附录原有的非常规拼写或音标标注，应以原书附录为准。
