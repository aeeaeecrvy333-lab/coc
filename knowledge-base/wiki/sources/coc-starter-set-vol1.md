## 概述

这份来源是《克苏鲁的呼唤》入门套装第一卷的中文 PDF，标题为《向火独行》。它不是常规的“KP 带队模组”，而是一篇面向单名玩家的段落式冒险：玩家一边阅读条目、一边创建角色、一边学习检定、理智、战斗与法术等基础规则。

与第三卷那种偏向“守秘人如何开跑”的新手模组不同，第一卷的重点是让玩家独自通过一次完整、危险、可反复尝试的故事，直观理解《克苏鲁的呼唤》的体验结构。它既是教程，也是正式的恐怖冒险。

## 核心内容

### 引言部分

- 用非常简短的方式解释《克苏鲁的呼唤》是什么、怎么掷骰、什么叫模组。
- 明确告诉读者：可以先玩完《向火独行》，再系统通读第二卷规则。
- 强调这是一篇“你自己一个人就能玩”的特制冒险。

### 向火独行

- 设定在 1927 年，玩家在前往阿卡姆赴任的途中，被长途车司机西拉斯半故意地滞留在烬头村。
- 角色会在故事开头现场分配属性，并在与西拉斯的交谈中，从 5 个职业里选择其一：文物学家、医生、记者、私家侦探、教授。
- 冒险主体分成“初到村庄 -> 第一天白天调查 -> 第一夜异变与警告 -> 第二天再调查 / 逃跑 -> 灯塔献祭高潮”几大段。
- 故事核心不是传统破案，而是识破烬头村以“节日”之名维持的献祭仪式，并在有限时间里逃出生天，或在绝望中接触天火相关的神话力量。
- 模组通过条目跳转逐步教学：属性检定、技能检定、奖励/惩罚骰、孤注一掷、战斗、理智损失、急救、技能成长、法术与魔法值消耗。
