## 概述

这份来源是《克苏鲁的呼唤》入门套装第三卷的中文 PDF，标题为“新手模组”，实际封面元数据为《追书人和其他冒险》。它主要面向新手守秘人与玩家，提供模组导入、预设调查员、展示材料使用方式，以及三篇入门冒险的正文或元信息。

与知识库既有四大规则来源不同，这份 PDF 不是规则总书，而是偏“实战开团”的冒险卷。它的价值主要在于补充低门槛模组、调查流程示范、1920 年代社会氛围，以及如何把调查员自然地卷入神话事件。

需要特别注意的是：这份本地 PDF 并不完整。《追书人》与《死者的顿足舞》有可用正文，《黑暗边缘》在本 PDF 中只有标题页、适玩人数与时长说明，并标注“待交稿……”。

## 核心内容

### 引言与使用方式

- 强调守秘人应先完整通读模组，再带玩家入场。
- 建议按“追书人 -> 黑暗边缘 -> 死者的顿足舞”的顺序游玩。
- 明确展示材料应在剧情指定时机交给玩家。
- 给出 5 名预设调查员，并把他们放进“未解之谜探索协会（SEU）”这一大学社团框架中。

### 追书人

- 标准的新手单人调查模组，适合一名守秘人配一名玩家，也可微调为两名调查员。
- 核心不是战胜怪物，而是发现“失踪叔叔其实自愿成为半食尸鬼”的真相。
- 模组用多个低风险调查点教学：邻居访谈、墓地调查、警方与报纸线索、监视、交谈或冲突分支。
- 结尾允许多种结果，包括理解、放手、误杀、见证群尸鬼离去等，带有明显的洛氏苦涩味。

### 黑暗边缘

- 该 PDF 仅提供一页元信息：适合 2-5 名玩家，建议 1-3 次游戏聚会。
- 标题页下直接写有“待交稿……”，说明本地 PDF 未收入该模组正文。

### 死者的顿足舞

- 强烈依赖 1925 年纽约哈莱姆区的种族、阶层、地下酒吧与爵士文化背景。
- 以勒罗伊·特纳被诅咒的小号为核心道具；该小号由奈亚拉托提普伪装成路易斯·阿姆斯特朗转交。
- 模组前半是多线调查：枪击、死者复起、黑帮、葬礼、乐手访谈、警方与报纸线索。
- 模组后半转向高潮追逐：黑帮绑架特纳、车库试验复生、墓园中以音乐唤起群尸。
- 主题上同时处理宇宙恐怖、黑人城市文化、制度性种族主义与“以艺术召来死亡”的悲剧感。
