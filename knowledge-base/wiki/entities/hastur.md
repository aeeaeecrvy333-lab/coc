## 基本信息

它的统治超越了人类认知，不与人类同在却又永远近在咫尺，随时准备腐化、玷污并耗空万物。梦中所见的它，总是臃肿、潮湿、无骨、触手横生，像一团永不安息的污浊水体伏在视线边缘发出呼唤。

别名：阿扎托，讳名者，不可言说者，扎斯特尔。不可言说者据说居于金牛座中的毕宿五近侧。许多资料把它与[[cthulhu.md|克苏鲁]]联系起来: 有人说两者是“兄弟”，有人认为二者彼此敌视，还有一种更怪异的学说认为它们其实是同一存在分裂出的两半。无论哪种说法为真，哈斯塔都被描写为一位高度腐化性的支配者，其存在本身就足以让文明、道德与感官逐渐溃烂。

《死灵之书》宣称直呼其名便会吸引其注意，也有文献认为它与[[yog-sothoth.md|犹格-索托斯]]、外神以及旧神之间都保持着复杂而敌对的关系。比较稳定的一点是：哈斯塔并非只以单一形态存在。人类文献中提到的黄衣之王、琥珀长者与劫掠者，都被视为这位不可言说者的外延、化身或意志投影。

它似乎也像许多其他高位存在一样受到旧神束缚。传说哈利湖既是它的居所，也是它的牢笼；卡尔克萨与毕宿五，则像是其影响的不同显现界面。

## 详细设定

### 教团

哈斯塔教团在地球上根基深厚，但大多数组织都保持小团体、孤立主义的结构。这样既让信徒专注于单一目标，也降低了整体暴露的风险。不同团体的风格差异极大: 有些原始血腥，有些则高度知识化、仪式化。很多人并不直接崇拜“哈斯塔”本体，而是通过某个更可理解的化身入手，比如[[byakhee.md|拜亚基]]相关的教派、琥珀长者的启示团体，或黄衣之王的颓废艺术社群。

这些教团的行动通常可以分为三类:

- 在世界中散播腐化萌芽，为神祇回归铺路。
- 获取知识与法术，争取在神祇身侧获得一席之地。
- 寻找方法削弱束缚、打开通向哈利湖或卡尔克萨的通道。

### 可能的赐福

- 异星智慧：灌输克苏鲁神话技能与相关法术，例如拜亚基召唤、黄金蜂蜜酒酿造等。
- 结盟：与其仆从、眷族或附庸建立联系，甚至发生精神互换。
- 不可言说的誓约：最受青睐者会与神祇建立不可逆的依附关系，最终成为完全附庸。
- 消解骨骼：少数信徒可让自己化为柔软血肉，更易穿过障碍与规避束缚。

### 遭遇

真正直面哈斯塔本体的情况极为罕见，更多时候调查员面对的是它的仆从、教团或已经萌发的腐化效应。某些典籍、塑像与画作会像在对读者“说话”，许诺知识与力量，诱使人接受誓约。随着精神连接建立，腐化会先侵蚀身体与感官，再侵蚀意志。

### 氛围

接近哈斯塔与其仆从时，人会不由地产生一种滑腻、潮湿、腐败的感官错觉，仿佛深水与腐肉在意识边缘交叠。最终，所有情感都可能被一种对生命本身的厌恶所压过。

### 理智损失

理智损失：`1D10/1D100` 点理智值。

### 魔法

- 意志：175
- 魔法值：35
- 法术：真实一瞥授予术、瘟疫发生术、青春吸取术、液化术、哈斯塔之歌、拜亚基召唤术、腐朽之触、不可言说的誓约等，以及其他守秘人决定的法术。

### 能力

- 躯体改换：其实体形态并不固定，可任意拉伸、缩回、扭曲或重组肢体，甚至临时呈现更接近目击者的形态，再加以恐怖化。
- 腐化：无论是当面遭遇还是精神接触，只要意志不足，便可能被植入腐化萌芽。萌芽会在数日到数年后爆发，让宿主疲惫、血肉溃烂、理智流失，并在最终诱导其接受“不可言说的誓约”。
- 直呼吾名：出声说出“哈斯塔”这个名字，有概率引来它的注意。结果可能包括短暂显现、噩梦、厄运缠身、拜亚基来访，甚至被直接拖入神祇的注意范围。
- 扭曲感官：接触哈斯塔的人类会发现颜色、声音、气味与空间感开始扭曲，未通过极限难度意志检定者在多数行动上承受惩罚骰。
- 显现于地球：其实体通常只能在毕宿五位于地平线以上的时段被召唤至地球，但精神显现随时可能发生。

它每轮可以用利爪与触手攻击选定目标三次，同时还会用额外触手在近距离内随机抓住最多 `1D3` 人。被抓住的人会受到挤压或被短暂控制后再遭抛出。

- 30 点跨维度血肉。
- 每轮恢复 `5` 点生命值。
- 若生命值降至 `0`，它会炸裂并喷出恶臭污物，然后在 `2D10` 小时后重新成形。

## 化身

### 琥珀长者

琥珀长者是类人灵体，披着琥珀色长袍，露出豆荚状躯体，额头仅有一只黄色独眼。它更像信使、喉舌与启示者，而不是专门传播卡尔克萨腐化的执行者。传说它居于“琥珀密室”，这座神殿可能位于隆高原、幻梦境，或根本不固定于任何一处。

它可能通过附身接受过“不可言说的誓约”的人类来显形，也可能直接出现在某个即将发生灾难的地方，为被选中的人带来警告、知识或试炼。

| 属性 | 数值 |
|------|------|
| STR | 100 |
| CON | 100 |
| SIZ | 70 |
| DEX | 100 |

- 生命值：17
- 伤害加值：+1D6
- 体格：2
- 移动：12

攻击方式：

- 每轮 `2` 次触手攻击，或 `1` 次拥抱 / 精神震爆。
- 拥抱（战技）：抓住目标后，每轮吸取 `1D10+5` 点体质与 `1D4` 点生命值。
- 精神震爆：对 `5` 码/米范围造成 `1D10` 点伤害，极限难度意志检定可减半。

护甲：

- 无。
- 若生命值降至 `0`，该化身会碎裂成尘，并在 `24` 小时后于哈斯塔意志选定之处重新现形。

### 劫掠者

劫掠者通常被理解为哈斯塔的怒火、报复与追猎本能的实体化。它会被派去杀死、惩罚或吞食那些违逆、烦扰或伤害不可言说者的人。关于它最常见的记述，是与噩梦、寒风和“被追猎感”一同到来。

| 属性 | 数值 |
|------|------|
| STR | 245 |
| CON | 500 |
| SIZ | 160 |
| DEX | 100 |

- 生命值：66
- 伤害加值：+4D6
- 体格：5
- 移动：10/30 飞行

攻击方式：

- 每轮 `1` 次触手击打，或 `1` 次吞噬大脑。
- 触手击打：用 `2D10` 根刀刃状触手集中攻击单个目标，每根造成 `1` 点伤害。
- 吞噬大脑（战技）：缠住目标后，下一轮开始钻挖头颅，先造成 `1D6` 点伤害并吸走 `1D10` 点智力，之后每轮继续撕裂智力，直至受害者死亡。

护甲：

- 20 点非地球皱皮。
- 若生命值降至 `0`，劫掠者会炸裂并在 `1D100` 天后于哈斯塔意志下重新现形。

### 黄衣之王

黄衣之王既可被视为哈斯塔最重要的化身，也可被视作一个已在某种程度上生出“自由意志”的独立存在。它与卡尔克萨、黄印、戏剧文本《黄衣之王》以及颓废、腐化、美感和启示紧密相连。许多信徒并不认为自己在崇拜哈斯塔，而是直接向黄衣之王献身。

它身着黄色、橙色与棕色的长袍或褴褛衣带，看起来更像是由这些“袍服”组成，而不是穿着它们。苍白面具之下可能是真脸，也可能是无数微小伪足。黄印则是其最著名也最具感染性的符号之一。

| 属性 | 数值 |
|------|------|
| STR | 125 |
| CON | 530 |
| SIZ | 70-100 |
| DEX | 135 |

- 生命值：60
- 伤害加值：+1D6
- 体格：2
- 移动：15（可任意出现/消失）

攻击方式：

- 每轮 `6` 次长袍攻击，或 `1` 次凝视 / 面部伪肢“亲吻”。
- 亲吻（战技）：附肢快速伸出并抓住目标，每轮吸取 `1D10` 点 `POW`；若对附肢造成足够伤害，可将其切断。
- 黄衣之王的凝视：聚焦某一受害者时，目标每轮失去 `1D6` 点理智值，除非通过极限难度意志检定躲开其注意。

护甲：

- 无。
- 若生命值降至 `0`，黄衣之王会在炫目的黄白闪光中消失，并在 `1D100` 天后于卡尔克萨王座上重现。
