# 模组：猩红文档

## 基本信息

- **模组名称**：猩红文档 (Crimson Letters)
- **时代背景**：1920 年代中某一年的 10 月，禁酒令时期
- **地点**：阿卡姆及其周边地区
- **适合人数**：2-5 名调查员
- **游戏时长**：1-3 次游戏聚会
- **模组定位**：开放式调查模组；凶杀、失踪文档、嫌疑人博弈与神话怪物并行推进
- **建议调查员**：最好已有团队关系，具备社交、图书馆使用、心理学、侦查或学术背景会更顺手

## 第一层：Wiki概览

### 模组用途

《猩红文档》不是给 KP 一路照章念剧情的，而是训练这些能力：

- 如何主持开放式调查
- 如何让多个嫌疑人都“像真凶”
- 如何把地点、人物欲望和怪物威胁拧成同一件事
- 如何在玩家自由调查时维持节奏，而不是等他们自然撞上结局

### 一句话前提

密斯卡托尼克大学的查尔斯·莱特教授离奇死亡，一批和阿卡姆女巫审判有关的珍贵文档随之失踪。法伦主任雇佣调查员找回文档并控制丑闻；但案子背后既有莱特的造假与赌债，也有多位危险买家和觊觎者，更有一个从文档中被放出来的怪物正在逼近完全显形。

### 守秘人真相

- 莱特是个前程看好的学者，但也长期伪造与盗卖文物、赌博欠债
- 他在评估霍布豪斯遗产时发现了一套价值极高、且带有神话危险的女巫审判文档
- 在试图复制、出售、周旋于多方买家时，文档中的存在被释放
- 莱特最终被怪物逼死，文档则落入守秘人选定的某位真凶或窃贼之手
- 这篇最重要的准备工作，不是背地图，而是 **先决定是谁拿走了文档、他们现在想做什么**

### 核心冲突

- 谁才是偷走或夺走文档的人
- 调查员会先相信谁，又会得罪谁
- 他们是否把注意力放在“文档去向”还是“怪物威胁”上
- 他们能否在怪物完全显形前，处理掉最危险的局面

### 结构总览

推荐把模组理解成四个阶段：

1. **案情建立**：受雇、看尸体、确认失踪文档与莱特的双面人生
2. **嫌疑人网**：法伦、考特、洛奇、弗林德斯、露西、维克、黑帮、亨特
3. **地点推进**：莱特小屋、校内办公室、酒馆、古董店、霍布豪斯宅邸、疗养院等
4. **危机收束**：文档现身、怪物显形、谁会死、谁会抢、最终如何收

### 主要线索点

- **莱特尸体与办公室**：案件起点，确认异常死亡
- **莱特小屋**：双重生活、造假、藏匿、赌债与私人物证
- **露西·斯通**：莱特私人生活、赌债、买家、亨特与维克
- **亨特 / 疗养院**：复制文档、精神崩溃、怪物首次大规模显形
- **维克 / 古董圈**：文档价值、危险买家、非人威胁
- **霍布豪斯宅邸**：文档历史与怪物根源
- **黑帮**：给模组加入肉身威胁与外部施压

### 结局走向

- **找回文档并压制怪物**：最佳结局
- **只找回文档，怪物仍活跃**：表面成功，后患无穷
- **怪物彻底显形**：阿卡姆本地惨剧
- **文档转手，危险外流**：长期坏结局

## 第一层半：背景故事与戏剧核心

### 背景故事

查尔斯·莱特不是个纯粹倒霉的学者。他是一个很聪明、很体面、也很会把自己卖得像成功人士的人。但在那层学术和社交外壳之下，他长期靠伪造、倒卖和赌博维持自己的生活。霍布豪斯遗产里的那批女巫审判文档，对他来说本来只是又一次捞钱机会。

坏就坏在，这次他碰到的不只是值钱古物。文档本身带着神话危险，而他为了复制、转卖、抬价，把文档拆开、交给亨特仿造，又向一群根本不该打交道的人放出消息。于是案子一层层坏下去：买家闻风而来，黑帮债主跟上，亨特先疯，莱特再死，文档失踪，而真正被放出来的东西则沿着所有碰过文档的人慢慢逼近。

所以这篇真正的故事不是“谁杀了莱特”那么简单，而是“一个以为自己能把一切都包装成交易的人，最终把交易做到了人类理解力之外”。而玩家来到这里时，残局已经散在一整张人际网里。

### 这篇真正的乐趣在哪里

- 它不是传统侦探推理，而是“人心、利益和神话威胁一起撒开的残局”
- 所有 NPC 都像在说真话，但没人会把全部真话交出来
- 真相不是一把钥匙，而是一堆会彼此咬合的危险动机

### KP 该把它讲成什么味道

- 表面要像大学命案与文档丑闻
- 中段要像越挖越脏的阿卡姆人物网
- 后段要让玩家意识到：他们不只是在破案，而是在追一场已经失控的释放过程

### 这页的使用边界

这页更适合作为 **开放调查模组的收束控制提纲**，而不是把所有 NPC、地点和嫌疑链都一股脑跑完的“全清单攻略”。《猩红文档》最容易失控的地方，不是线索太少，而是线索太多、每个人都还能继续深挖。

因此，KP 实跑时不需要把每一名嫌疑人、每一条副线、每一处地点都完整跑满。只要玩家已经同时掌握下面三类信息中的两到三类，这篇就应开始往高潮收束，而不是继续横向扩写：

- 莱特生前的双重生活与文档交易链
- 文档现在大概率落在谁手里，或至少落在哪一圈人手里
- 亨特、维克或其他接触者已经证明怪物威胁正在升级
- 再拖下去，局面会从调查转成争抢、逃跑、死亡或显形

这页真正要帮助 KP 守住的，不是“把所有内容都展示出来”，而是知道什么时候信息已经够了，应该让人开始动、让怪物开始逼近、让文档与真凶在同一时刻撞上玩家。

## 第二层：KP 实跑流程

## 开跑前一定要先做的事

### 1. 先选“谁拿走了文档”

这是整篇最关键的 KP 决策。最适合的几个候选：

- **安东尼·弗林德斯**：偏学生痴迷与崩溃路线，调查风味最浓
- **阿伯那·维克**：偏危险黑幕与非人势力路线，威胁最大
- **艾米莉亚·考特 / 哈兰德·洛奇**：偏学术嫉妒、人心与算计路线
- **黑帮 / 第三方买家**：偏外部夺取与追逐路线

默认推荐：

- 想要“开放但仍好控”时，选 **弗林德斯**
- 想要“真正危险、后劲十足”时，选 **维克**

### 2. 再决定怪物显形速度

这篇不好跑的原因之一，是怪物太容易被忘掉。开团前建议直接定一个隐性时钟：

- **慢速**：前半几乎只留迹象，后半才杀人
- **标准**：每过一段调查时间、每有关键人触碰文档，怪物升级一次
- **快速**：玩家拖得太久就直接死人、附身或显形

### 3. 记住这篇不是一本道

KP 不需要把玩家拉回唯一正确路线。你要做的是：

- 让每个 NPC 都提供一部分真相
- 让每个地点都至少能把玩家导向两个人或两个地点
- 当玩家走偏时，不是“说你们错了”，而是让世界继续动起来

## KP 快速跑团卡

### 一句话推荐推进链

法伦委托 -> 看尸体 / 办公室 -> 锁第一批嫌疑人与地点 -> 莱特小屋 / 露西 / 亨特 / 维克等地点推进 -> 判断文档流向 -> 怪物升级 -> 高潮收束

### 绝不能漏的 5 条线索

- 莱特不是单纯受害者，他有 **双重生活**
- 文档失踪和 **复制 / 造假 / 买家竞价** 紧密相关
- 亨特的崩溃说明文档里放出来的东西 **是真的**
- 维克、黑帮、露西、校内人各自都在推动局面
- 这篇不能只查“谁偷了文档”，还得盯 **怪物什么时候失控**

### 玩家卡住时最自然的强推句

- 法伦推起手：`“去查莱特死前最后见过谁、碰过什么。”`
- 露西推私生活线：`“莱特有些朋友，可不适合见光。”`
- 小屋推买家线：`“他不是被一个人盯上，而是被一群人盯上。”`
- 亨特推神话线：`“他疯成这样，不会只是因为钱。”`
- 怪物推高潮：`“你们已经没空继续慢慢盘问了，事情开始自己往外流血了。”`

### 新手 KP 最容易漏掉的事

- 不要等所有地点都跑完才收束
- 一旦玩家锁到几条关键线，就该让人开始动、开始跑、开始死
- 这篇不是线性模组，重点是“什么时候够了”
- 真凶是谁要提前决定，但不要提前演成唯一答案

## 一页式线索表

| 线索簇 | 常见获取处 | 指向哪里 | 没拿到时怎么补发 |
|------|------|------|------|
| 莱特死得不对 | 尸体 / 办公室 / 法伦 | 这不是普通心脏病案 | 惠特克罗夫特或校内传言补发 |
| 莱特有双重生活 | 莱特小屋 / 露西 / 黑帮 | 私生活线、赌债、买家线 | 小屋没去成就让露西直接补 |
| 文档曾被复制和造假 | 亨特 / 露西 / 小屋物证 | 亨特主线、怪物释放原因 | 露西或校内记录都能补 |
| 不止一个人盯上文档 | 维克 / 黑帮 / 露西 / 弗林德斯 | 多方势力并行 | 任意两方碰头就该让玩家明白 |
| 霍布豪斯文档本身有问题 | 霍布豪斯宅邸 / 维克 / 亨特疯话 | 神话危险线 | 维克或亨特都能补一部分 |
| 亨特疯掉不是单纯压力大 | 疗养院 / 疯话 / 怪物迹象 | 怪物已被释放 | 若没见亨特，就用怪物袭击别人补 |
| 露西处境危险 | 露西 / 黑帮监视 / 怪物追踪 | 外部压力升级 | 让她失踪、来求助或被尾随来补 |
| 维克不只是普通古董商 | 维克店 / 露西 / 地下圈消息 | 更大更黑的危险 | 若玩家不碰维克，就让维克主动碰玩家 |
| 文档正在某人手里移动 | 真凶行动 / 交易 / 逃跑 / 仪式 | 高潮收束 | 玩家拖太久就主动触发 |
| 怪物已从迹象进入杀意 | 受害者新增 / 附身 / 显形 | 该收束，不该再无限查 | 用一次袭击强制补发 |

## 新手 KP 先看：推荐推进链怎么用

这篇不是严格线性模组，所以不要硬写成“场景一接场景二”。新手 KP 最稳的用法是把它看成 6 个调查区块。

| 当前区块 | 玩家通常为什么会来 | 这一段拿到什么就够了 | 下一段最自然怎么接 |
|------|------|------|------|
| 受雇与案情建立 | 开场默认进入 | 莱特死得不对，文档丢了 | 进开放调查网 |
| 开放调查网 | 想锁嫌疑人 | 锁到 3-5 个关键人和对应地点 | 去具体地点 |
| 地点推进 | 想把嫌疑落到实处 | 莱特双重生活、买家、亨特、黑帮、霍布豪斯线成形 | 锁定文档去向 |
| 嫌疑人运作 | 玩家开始盯某几个人 | 判断谁在说谎、谁在动、谁会先出手 | 怪物时钟升级 / 高潮逼近 |
| 怪物时钟 | 拖久了或有人碰文档 | 意识到这不是普通命案，而是持续升级的超自然事件 | 进入高潮 |
| 收束与高潮 | 玩家锁到文档或某人要跑 | 文档、真凶、怪物在同一时刻撞上 | 结尾 |

## 第一幕：受雇与案情建立

### 这一幕的故事作用

给玩家一个看似体面的入口，同时把“这事远没有校方说得那么干净”钉住。

### 这一幕的情绪重点

压着嗓子说话的丑闻感。不是惊悚先来，而是体面先裂开。

### 玩家通常怎么来到这一幕

默认开场，没有额外条件。

### 这一幕什么时候算完成

只要玩家清楚下面三件事，这幕就完成：

- 莱特死了
- 文档丢了
- 校方想压事，但事情并不干净

### 这一幕之后怎么推到下一幕

给他们第一批人和地点，然后直接放进开放调查网。

### KP 开场描述

> 密斯卡托尼克大学里弥漫着一种压着嗓子说话的紧张气氛。教授死亡的消息已经传开，但没有人肯把话说满。走廊里每个人都像知道一点什么，又都怕把它说出来。法伦主任找上你们时，态度客气，眼神却比礼貌更急。

### 法伦的作用

法伦不是单纯的发任务 NPC，他同时承担：

- 给调查员名义上的合法入口
- 提供报酬与时间压力
- 强调“最好别把丑闻闹大”

### 这一幕必须让玩家知道什么

- 莱特已死
- 一批重要文档失踪
- 校方最在意的是尽快找回文档并压下影响
- 死因表面上能解释，实际非常不对劲

### 第一批推荐投放的地点

- **看尸体**
- **莱特在校办公室**
- **法伦 / 惠特克罗夫特的进一步问话**

### 看尸体怎么跑

看尸体的作用是把“这不是普通心脏病”钉死。重点不是解剖细节，而是：

- 恐惧凝固在脸上
- 眼眶里的异样组织
- 靠近尸体就让人不舒服

### 玩家若问“现在第一步该查哪里”

KP 可以自然收束：

> 眼下最稳的两条路，一条是查莱特死前最后碰过的人和地方，另一条是从他失踪的文档入手。

## 第二幕：开放调查网

### 这一幕的故事作用

把案件从单一凶杀扩成一整张欲望网络，让玩家开始在“谁有动机”和“谁更危险”之间摇摆。

### 这一幕的情绪重点

别跑成清单式访谈。要有“每个人都在藏东西，每个人都想把你往别处带”的味道。

### 玩家通常怎么来到这一幕

他们从法伦、尸体、办公室线出来后，自然会开始问“那该先找谁”。

### 这一幕什么时候算完成

新手 KP 不要等全查完。只要玩家已经锁到下面任意 3 类对象，就够进入下一段：

- 校内人：法伦 / 考特 / 洛奇
- 私生活人：露西
- 外部危险人：维克 / 黑帮
- 崩溃关键人：亨特

### 这一幕之后怎么推到下一幕

把这些人导向具体地点即可。开放调查网的作用不是终点，而是发牌。

### 你应该怎么理解这一段

这里不是场景顺序，而是 **人物网 + 地点网**。KP 的任务不是逼玩家按顺序打卡，而是让他们每走一步都获得：

- 新嫌疑人
- 新地点
- 更高风险

### 先给玩家哪几个嫌疑人最顺

推荐优先放出这 5 个：

1. **法伦**：因为他雇人，也最着急  
2. **考特**：学术圈内部关系  
3. **洛奇**：竞争者气味最重  
4. **露西·斯通**：私人生活线  
5. **维克**：买家与危险感

### 嫌疑人不是单选题

这篇的手感来自：每个人都有动机，但动机不完全相同。

- 有人要钱
- 有人要名
- 有人要文档
- 有人要摆脱麻烦
- 有人想利用怪物

### KP 操作原则

- 不要让任何 NPC 一次把话说完
- 先给表层回答，再让心理学、追问或旁证撬出更深一层
- 某人说谎，不代表此人就是真凶；这点很重要

## 第三幕：地点怎么跑

### 这一幕的故事作用

让抽象的人物关系变成可踏进去的空间，让每个地点都像某类秘密的容器。

### 这一幕的情绪重点

不同地点要有不同质感：校内是体面裂纹，小屋是私生活崩坏，酒馆是地下气味，维克店里则是礼貌包着威胁。

### 玩家通常怎么来到这一幕

他们已经盯上某人，现在要去“那个地方看看”。

### 这一幕什么时候算完成

不要求他们把所有地点都跑完。只要地点调查让下面两件事成立，就可以继续：

- 玩家已经能大致判断谁更可疑、谁更危险
- 玩家开始锁到文档流向或怪物后果

### 这一幕之后怎么推到下一幕

地点会把玩家再推回“嫌疑人运作”和“怪物时钟”：有人要跑、有人要抢、有人要灭口、怪物也开始升级。

## 莱特的小屋

### 这地方负责什么

- 证明莱特的双重生活
- 把学术案件拉进造假、赌债和情人世界
- 给露西、黑帮、维克与造假链条落地

### 玩家在这里应能得到

- 莱特有隐藏财物或藏匿习惯
- 他和不干净生意有关
- 他死前处于被人追、被人盯、急于跑路的状态

### 常见现场事件

- 黑帮找上门
- 别的调查者或盯梢者出现
- 露西先来翻东西

## 莱特在校办公室

### 作用

- 让玩家感觉“死亡就在这里发生”
- 给出学术世界的表面秩序与实际恐慌
- 补校内人物关系

### 这里要重点强调

- 这是上锁密室式死亡，不正常
- 莱特死前回来收拾值钱物件
- 有人想尽快清理痕迹

## 希布酒馆

### 作用

- 让露西线自然登场
- 让阿卡姆台面下的气息出现
- 引出黑帮监视和低层消息渠道

## 霍布豪斯宅邸

### 作用

- 让“女巫审判文档”从商品变回真正危险的历史遗物
- 提供怪物起源感
- 给最后的仪式或高潮场景留位置

## 轻率琐事（维克的店）

### 作用

- 这地方要有“太有礼貌，反而更危险”的感觉
- 维克若为真凶，这里可以是最终高潮场地
- 即使他不是真凶，这里也能让玩家意识到文档背后还有更大的世界

## 疗养院与亨特

### 作用

- 说明复制文档的人已经疯掉
- 把怪物第一次“真正动过手”的证据抛给玩家
- 为“文档一旦接触会带来什么”做最直观展示

### 亨特怎么演

亨特不是稳定线索 NPC，而是破碎的线索容器。不要把他演成谜语人，要演成：

- 时而清醒，时而语无伦次
- 特定词汇会刺到他，逼出零碎真相
- 玩家听完后觉得“他说得乱，但里面有真的东西”

## 第四幕：嫌疑人怎么用

### 这一幕的故事作用

不是列人物卡，而是提醒 KP：这些人都是推动剧情的发动机。

### 这一幕的情绪重点

谁都别演成“来送线索的功能角色”，他们都应该像有自己的账要算。

### 玩家通常怎么来到这一幕

不是显式进入，而是当玩家已经开始重点怀疑某几个人时，KP 就该切换到这套思路。

### 这一幕什么时候算完成

当你能明确回答下面两件事时，这段就完成了：

- 现在谁最像拿着文档的人
- 现在谁会先动手

### 这一幕之后怎么推到下一幕

用人的动作去推进怪物时钟或直接把剧情推入高潮，比如：

- 维克发邀请
- 黑帮上门
- 露西失踪
- 亨特逃逸

## 布莱斯·法伦

- 功能：发布委托、代表大学、制造体面压力
- 表层印象：克制、体面、想控场
- 深层用途：可做复杂雇主，也可做自保型嫌疑人

## 艾米莉亚·考特

- 功能：校内线、学术线、敏感细节
- 表层印象：聪明、紧张、知道得比说得多
- 深层用途：既可做危险知情者，也可做被怀疑的无辜者

## 哈兰德·洛奇

- 功能：竞争对手感最强的人物
- 表层印象：不太讨喜，很容易被玩家盯上
- 深层用途：适合当假真凶，用来吃玩家早期怀疑

## 安东尼·弗林德斯

- 功能：痴迷、跟踪、学术狂热、崩溃边缘
- 表层印象：看起来很像会惹出事的人
- 深层用途：默认最佳真凶人选之一

## 露西·斯通

- 功能：进入莱特私人生活的最好入口
- 表层印象：精明、脆弱、现实
- 深层用途：高危证人；谁都可能想利用她或灭口她

## 阿伯那·维克

- 功能：把模组从学园命案拉到神话黑市层面
- 表层印象：稳重、优雅、危险
- 深层用途：最强势的真凶或终局操盘手

## 奥克斯与肖纳西

- 功能：给这篇加入可以落地的肉身威胁
- 表层印象：讨债的，粗暴但不傻
- 深层用途：逼玩家快一点，也能制造伏击和误会

## 第五幕：怪物时钟

### 这一幕的故事作用

保证模组不会塌成纯社交悬疑，持续提醒桌上：真正更坏的东西还在逼近。

### 这一幕的情绪重点

怪物前期最好像一阵越来越难解释的风，而不是随叫随到的战斗单位。

### 玩家通常怎么来到这一幕

不是主动来，而是你作为 KP 应该在调查推进中持续暗中运转它。

### 这一幕什么时候算完成

只要怪物已经从“迹象”推进到“明确事件”，就该考虑把模组收束，不再无限调查。

### 这一幕之后怎么推到下一幕

用一次真正的危机把故事推入高潮：

- 某人被袭击
- 某人带文档逃跑
- 某地开始举行仪式
- 怪物局部或完全显形

### 千万别忘了怪物

这篇最容易跑坏的点，就是 KP 一直沉浸在人物关系里，结果怪物像没来过。建议你在桌上暗记一个升级表。

### 推荐的 4 级推进

1. **不安迹象**：风声、墙内声、被注视感、轻微异常  
2. **部分显形**：有人看见诡异影子、局部现身、附身边缘  
3. **明确杀意**：攻击证人、追逐文档接触者、逼疯亨特式事件再现  
4. **完全显形**：在阿卡姆某处造成公开惨剧

### 什么会推动时钟

- 玩家调查拖太久
- 更多人碰触文档
- 真凶试图利用文档
- 玩家把文档交给以为安全的人

## 第六幕：收束与高潮

### 这一幕的故事作用

把文档、真凶、关键证人和怪物威胁压缩到同一时刻，逼出真正结局。

### 这一幕的情绪重点

要有“所有还没算完的账一起到了”的感觉。不是单点爆炸，而是多方同时合拢。

### 玩家通常怎么来到这一幕

常见触发条件有三个，满足其二就够：

- 玩家基本锁定文档位置
- 某关键 NPC 要跑路 / 交易 / 仪式
- 怪物升级到不能再放着不管

### 这一幕什么时候算完成

下面三件事里，只要桌上明确了两件，就可以收尾：

- 文档最后落在谁手里
- 真凶 / 真窃贼是谁
- 怪物有没有被压住

### 这一幕之后怎么推到下一幕

没有下一幕，直接结算成最佳、表面成功或坏结局。

### 什么时候该进高潮

当下面三项里任意两项成立时，就可以把模组推入高潮：

- 玩家已基本锁定文档位置
- 怪物升级到明确杀人或附身
- 某关键 NPC 要跑路、交易、仪式或灭口

### 常见高潮地点

- 维克的店铺地下
- 霍布豪斯宅邸
- 亨特公寓 / 疗养院逃逸现场
- 某次文档交接地点

### 玩家在高潮里可能做什么

- 抢文档
- 救人
- 打断仪式
- 与怪物正面冲突
- 带着文档逃

### KP 需要做的不是“安排唯一解”

而是判断：

- 文档现在在谁手上
- 谁最想活
- 谁最想抢
- 怪物此刻最容易攻击谁

这样高潮自然会长出来。

## 结尾怎么收

### 最佳结局

- 文档被妥善控制或封存
- 怪物被驱散或未能完全显形
- 主要无辜者幸存
- 调查员拿到钱，也意识到自己碰到了比钱大的东西

### 表面成功结局

如果玩家把文档交回去了，但怪物没处理掉，可以让第二天报纸写出另一场惨剧，提醒他们事情并没完。

### 坏结局

若怪物在阿卡姆市内完全显形，应该让它像一次真正的城市丑闻和恐怖传闻，不只是战斗失败。

## 奖励

- 成功归还文档，可获得报酬，或每人约 **250 美元**
- 若成功战胜怪物，每人获得 **1D6 点理智值**
- 若怪物完全显形且玩家未能阻止，按来源中的理智损失处理

## KP 兜底提示

### 玩家一直抓不住重点

用三类东西拉回：

- 新死者或新袭击
- 某个 NPC 主动来接触调查员
- 某处出现“有人正在拿文档做事”的新动作

### 玩家只盯一个嫌疑人

让别的线自己撞上来。比如：

- 黑帮找上门
- 露西失踪
- 亨特发病或出逃
- 维克发出礼貌邀请

### 玩家太快猜对真凶

不要否认他们。改为把压力转到：

- 怎么拿到文档
- 怎么阻止怪物
- 怎么在别的势力介入前收尾

### 玩家一直不敢碰文档

很好。让他们意识到这是明智的，同时也让别的 NPC 去碰，给他们看后果。

## 常见玩家问题与 KP 回答

### “法伦是不是在利用我们？”

> 他当然在利用你们，他花钱雇你们就是为了这件事。问题在于，他隐瞒了多少。

### “莱特到底是受害者还是活该？”

> 两者都可以是真的。他确实把自己卷进了不该碰的东西里，也确实因此送了命。

### “真凶一定是校内人吗？”

> 不一定。眼下所有和文档有接触的人，都有可能把事情做得更糟。

### “我们先把文档烧了行不行？”

> 这可能是个思路，但你们最好先弄清楚它现在在哪、有没有人拿它做别的事，以及烧掉它会不会太晚。

## KP 现场可直接说的句子

### 法伦委托时

> “我需要的是谨慎、效率，还有分寸。尤其是最后那个。”

### 看到莱特尸体时

> “这不是单纯死于惊吓的人会留下的表情，更像是他直到最后一刻都在看着什么。”

### 维克登场时

> “他对你们很客气，客气得像已经想好了你们该为他做什么。”

### 露西说起莱特时

> “他爱钱，也爱自己看起来像个不会失败的人。”

## 常见分支处理

### 玩家想把所有嫌疑人都叫来对质

可以，但这更像制造局面，而不是自动破案。真正有用的是让不同人彼此看见、彼此心虚、彼此抢话。

### 玩家想报警

警察会处理治安和死人，不会替他们理解神话文档与怪物。过早报警反而可能让事情更难控制。

### 玩家想把文档立刻交回大学

可以，但如果怪物尚未处理，等于把风险转移给法伦和校园。
