## 基本信息

无形马是来自另一存在位面的半实体生物种族。在本位面中，人类肉眼通常无法看见它们，但它们也可以显现，其形象如同苍白的幽灵马，有着前腿、前蹄及硕大的头部，后部则模糊不清，或萦绕在怪异的厚重雾气中。通常情况下，它们存在的唯一踪迹便是令人毛骨悚然的嘶鸣声与马蹄声。

这些生物在非实体形态下可以操纵物体，也可以穿过实体。它们若不显现为半实体化的幽灵马，便无法进行物理攻击。无形马通常独居，来到这一位面主要有两种可能：被某人召唤，或被极度的痛苦吸引而来，例如可怖谋杀、自杀或酷刑发生之地。由此，它们常被误认作不得安息的鬼魂。

### 特殊能力

- 念力：在隐形的非物质形态下，无形马每消耗 1 点魔法值，便能推动具有 1STR 的人或物体。
- 穿透实体：可以穿过物件、墙壁与人类，但这样做会使其移动速度减半。
- 显现为马：若要显现为实体，无形马需要为每 5 点 SIZ 付出 1 点魔法值；实体化需要 1 轮时间。

若在其实体形态下生命值降至 0 点，无形马将被驱赶回躲藏处，直到下一次日落后才能完全恢复。若要真正杀死无形马，必须将其 POW 降至 0 点。

## 无形马，幽幻的似马生物

若化为实体，无形马拥有下列属性。在非实体形态下，它只有 INT、POW 与 DEX。

| 属性 | 掷骰 | 均值 |
|------|------|------|
| STR | (4D6+16)×5 | 150 |
| CON | (4D6+10)×5 | 120 |
| SIZ | (4D6+20)×5 | 170 |
| DEX | (4D6+6)×5 | 100 |
| INT | (2D6+10)×5 | 85 |
| POW | SIZ×1.5 | 255 |

平均生命值：29  
平均伤害加值（DB）：+3D6  
平均体格：4  
移动：14 / 20（非实体）  
平均魔法值：51

### 战斗方式

每轮攻击次数：2（猛踢）或 1（啃咬）

无形马要发动物理攻击必须化为实体。它们会用蹄子猛踹敌人，或用牙齿啃咬敌人。啃咬致死的受害者会被咬成两截；目睹这种攻击，或发现遗骸，都会造成额外理智冲击。

| 攻击方式 | 技能值 | 伤害/效果 |
|----------|--------|-----------|
| 格斗 | 75%(37/15) | 伤害 1D6+DB |
| 啃咬 | 50%(25/10) | 伤害 1D4+DB |
| 闪避 | 50%(25/10) |  |

### 技能

N/A

### 护甲

一切物理攻击均只造成 1 点伤害；魔法武器能造成正常伤害。除非 POW 降低至 0 点，无形马总会在下一个日落时分完全恢复。

### 理智损失

- 遭遇实体化的无形马：0/1D6
- 听到无实体隐形马的可怖嘶鸣与马蹄声：0/1D2
