## 基本信息

修格斯主宰是体型较小、智力更高、拟态能力更强的修格斯变种。它们能模仿人类外貌与语言，潜入人类社会，直到猎物放下戒心之后再揭下面具，显露真实形态。

别名：掉包者、拟态者、变形者、修格斯王。

这些怪物通常把自己伪装成极度肥胖、缺少毛发的人类。它们狡猾、迷人、善于操纵，同时又带着挥之不去的不协调感。修格斯主宰偏爱独居与秘密捕猎，会迅速处理任何识破其真面目的人。

---

## 详细设定

相较于庞大的普通修格斯，修格斯主宰更像是“学会了做人”的异常后裔。它们能够在人类形态与真实胶质形态之间切换，但维持人类伪装需要持续的专注与自我控制；一旦措手不及、愤怒或分心，它们就可能当场“融化”回原形。

修格斯主宰以渗透、猎食与伪装为乐。它们似乎特别沉迷于人类理解恐惧的那一刻，并把这种情绪视作猎食体验的一部分。它们没有真正稳定的社会性，也不会自然死亡或繁殖，但有传言称少数个体会结成小规模团体，推动某种长远而隐秘的计划。

### 特殊能力

- **拟态**：修格斯主宰可伪装成人类，使用语言、姿态与社会身份接近猎物。
- **失控变形**：若在人类形态下遭到刺激、受伤过重或意志失衡，它们可能迅速现出原形。
- **吞噬**：修格斯主宰会抓住并吞入目标，以腐蚀性酸液和消化液分解其躯体。受害者每轮可进行一次 `STR` 对抗检定尝试脱身。与普通修格斯不同，修格斯主宰一次只能吞噬一个目标。
- **抗性与再生**：火焰和电击伤害减半；常规武器每次命中通常只能造成 `1` 点伤害；每轮恢复 `2` 点生命值。

## 修格斯主宰，臃肿的畸兽

注：由于维持人类拟态需要大量精神控制，人类形态的部分属性低于真实形态。

| 属性 | 人类形态 | 真实形态 |
|------|----------|----------|
| STR | (2D6+8)×5 [75] | (4D6+10)×5 [120] |
| CON | (2D6+6)×5 [65] | (6D6+10)×5 [155] |
| SIZ | (2D6+8)×5 [75] | (2D6+8)×5 [75] |
| DEX | 3D6×5 [50-55] | 3D6×5 [50-55] |
| INT | (2D6+6)×5 [65] | (2D6+6)×5 [65] |
| POW | (2D6+6)×5 [65] | (2D6+6)×5 [65] |
| APP | (1D6+3)×5 [30] | N/A |
| EDU | 3D6×5 [50-55] | N/A |

平均生命值：人类形态 `14` / 真实形态 `23`  
平均伤害加值（DB）：人类形态 `+1D4` / 真实形态 `+1D6`  
平均体格：人类形态 `1` / 真实形态 `2`  
平均魔法值：13  
移动：人类形态 8 / 真实形态 10 滑行

### 战斗方式

每轮攻击次数：2（徒手、武器）或 1（碾压、吞噬）

修格斯主宰可以在人类形态下像普通人一样拳打脚踢或使用武器，但它更偏爱在真实形态下压制猎物并直接吞食。

| 攻击方式 | 技能值 | 伤害/效果 |
|----------|--------|-----------|
| 格斗（人类形态） | 60%(30/11) | `1D6+2+DB` |
| 碾压（真实形态） | 80%(40/16) | `DB` |
| 吞噬（真实形态，战技） | 80%(40/16) | 吞入目标；每轮造成 `DB` 级消化伤害 |
| 闪避 | 30%(15/6) |  |

### 技能

取悦 `70%`，恐吓 `60%`，语言（多种）`40%`，说服 `60%`，潜行 `65%`，以及守秘人决定的其他社会或伪装相关技能。

### 护甲

无固定护甲，但具有火焰/电击减半、常规武器钝化与再生能力。

### 法术

知晓 `1D3` 个法术。

### 理智损失

遭遇一只真实形态下的修格斯主宰损失 `1D6/1D20` 点理智值；若亲眼见到其由人类形态崩解成真实形态，再额外损失 `1/1D3` 点理智值。
